# BOT COOL E-YEAH

## Description

Bot Cool E-Yeah is a whatsapp bot.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ node app.js
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

Scan Qr Code with your whatsapp.
